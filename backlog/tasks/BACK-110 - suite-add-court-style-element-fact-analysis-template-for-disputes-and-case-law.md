## Description
Parent epic: #97

## 문제
판사/분쟁 관점에서 요건사실, 인정사실, 미확인 사실, 증거, 법률효과가 분리되어야 한다. 현재는 조문/판례 확인 후 결론 흐름에 가깝다.

## 작업
- research-workflow 또는 litigation reference에 법원식 판단 템플릿 추가.
- 판례 출력에 사안의 유사점/차이점, 적용 한계, distinguishing 필드 추가.
- 테스트에 구조 검증 추가.

## 완료 기준
- 분쟁형 질문은 요건/사실/증거/미확인/잠정 결론이 분리된다.
- 판례 인용이 사건번호 확인에 그치지 않는다.
