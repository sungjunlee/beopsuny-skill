## Description
Parent epic: #97

## 문제
형사/제재 관점에서 현재는 형량·승패 예측 금지에 비해 압수수색, 임의제출, 진술, 내부조사, 증거보전 workflow가 부족하다.

## 작업
- enforcement/criminal response workflow 설계.
- 중대재해, 개인정보 유출, 담합, 영업비밀, 산업기술, 임금체불, 무허가 영업 시나리오 추가.
- 증거보전, 변호인 선임, 임직원 인터뷰, 보고라인 gate 명시.

## 완료 기준
- 실제 수사/조사 개시 상황에서 안전한 초기 대응 가이드를 제공한다.
- 형량/승패 예측은 계속 금지된다.
